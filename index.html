⚙️ Настройка совместного доступа (Firebase) — нажмите, чтобы развернуть
Сейчас правки сохраняются только в этом браузере. Чтобы несколько человек редактировали онлайн с любых устройств:
  1. Зайдите на console.firebase.google.com → создайте проект (бесплатно)
  2. Слева Build → Realtime Database → Create Database → регион europe → режим Test mode
  3. Project settings → General → блок «Your apps» → значок </> (Web) → зарегистрируйте → скопируйте объект firebaseConfig
  4. Вставьте его в этот HTML-файл в блок FIREBASE_CONFIG вверху скрипта, замените плейсхолдер
  5. Выложите файл на GitHub Pages — и делитесь ссылкой
Когда ключи вставлены — индикатор станет «онлайн», и правки будут видны всем.
