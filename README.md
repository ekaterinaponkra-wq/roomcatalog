# Каталог квартир

Онлайн-конфигуратор квартир платформенных решений.

## Структура
- `index.html` — сайт конфигуратора (открывается сам)
- `images/` — планировки квартир (88 PNG, apt_001…apt_088)
- `.github/workflows/deploy.yml` — автодеплой на GitHub Pages

## Деплой
Сайт разворачивается автоматически при каждом push в ветку `main`
через GitHub Actions. После первого запуска адрес:
`https://<username>.github.io/roomcatalog/`

Настройка (один раз): Settings → Pages → Source → **GitHub Actions**.

## Совместное редактирование
Для синхронизации админ-правок между участниками вставьте ключи
Firebase в блок `FIREBASE_CONFIG` в начале скрипта `index.html`.
Инструкция встроена в сам сайт (жёлтый блок сверху).
